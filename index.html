데이터 입력

입력 전용 이 페이지는 ChatGPT/Gemini가 추출한 CSV를 넣는 곳입니다. 입력 후 결과 확인은 상단의 데이터 확인 탭에서 분리해서 볼 수 있어요.

1. AI에게 데이터 추출 요청

현재 선택한 연도/월을 기준으로, 다른 AI 계정에서도 같은 형식의 CSV가 나오도록 고정 프롬프트를 복사합니다.

상단 메인 근무표만 추출 CSV 헤더 고정 d01~d31 날짜 컬럼 유지 불확실한 값은 확인필요
복사 후 ChatGPT/Gemini에 근무표 이미지를 첨부하고 그대로 붙여넣으세요.

2. 추출된 CSV 입력

AI가 반환한 CSV를 그대로 붙여넣거나, CSV 파일로 저장해서 업로드할 수 있습니다.

필수 헤더: year, month, person_name, d01~d31

OPTION
