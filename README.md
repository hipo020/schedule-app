# 근무표 정리함 v1.5 OCR 베타

매월 이미지로 공유되는 근무 스케줄표를 업로드하고, OCR로 날짜별 스케줄 코드를 자동 추출한 뒤 검수표에서 수정하여 일간/주간/월간/오늘 출근/휴무일/공유/엑셀로 정리하는 정적 웹앱입니다.

## Vercel 배포

아래 파일 4개를 GitHub 저장소의 최상단에 올려주세요.

```text
index.html
styles.css
app.js
README.md
```

Vercel 설정은 다음처럼 두면 됩니다.

- Framework Preset: Other
- Build Command: 비워두기
- Output Directory: `.`
- Root Directory: 비워두기

## 사용 흐름

1. `설정·이미지` 페이지에서 기준 연도/월과 내 이름을 입력합니다.
2. 스케줄표 이미지를 업로드합니다.
3. `자동 추출` 페이지로 이동합니다.
4. 표의 위에서 아래 순서대로 사람 이름을 입력합니다.
5. 노란 추출 영역이 날짜별 코드 셀만 감싸도록 X/Y/가로/세로 값을 조정합니다.
6. `자동 추출 시작`을 누릅니다.
7. 추출 결과를 확인하고 `추출 결과 검수표에 반영`을 누릅니다.
8. `검수표` 페이지에서 틀린 코드를 수정하고 저장합니다.
9. 홈, 일간, 주간, 월간, 오늘 출근, 휴무일, 공유·엑셀 페이지에서 확인합니다.

## 주의

- 이 버전은 OCR 베타입니다.
- 사진이 기울어져 있거나 글자가 작으면 일부 코드가 틀릴 수 있습니다.
- 자동 추출 후 검수표에서 한 번 확인하는 흐름을 권장합니다.
- OCR은 브라우저에서 동작하며 `tesseract.js` CDN을 사용합니다. 배포 후 사용자의 브라우저에서 인터넷 연결이 필요합니다.
