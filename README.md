# 근무표 정리함 v2.1

개인용 월별 근무표 정리 웹앱입니다. Supabase 로그인/저장 구조를 유지하면서, 이미지 OCR 대신 ChatGPT/Gemini가 추출한 CSV 데이터를 앱에 입력해 정리하는 흐름을 중심으로 다듬었습니다.

## 파일 구성

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

GitHub 저장소 최상단에 위 4개 파일만 업로드하세요.

## 주요 변경 사항

- 데이터 입력 탭과 데이터 확인 탭 분리
- 추출 프롬프트 복사 기능 유지
- CSV 불러오기 후 데이터 확인 화면으로 이동
- OCR은 `OCR 보조` 탭으로 유지
- OCR 기본 이름을 Culinary 근무표 기준으로 변경
- 주간 / 월간 / 휴무일 UI 개선
- 공유·엑셀 화면의 카톡용 텍스트에 날짜별 출근 현황 우선 표시
- 오늘이 현재 월에 포함되어 있으면 실제 오늘 기준으로 출근 현황 자동 표시
- 현재 월이 아니면 선택한 날짜 기준으로 표시

## 기본 이름 목록

이준호, 류선협, 이상민, 김도영, 이승호, 곽병우, 이미현, 이다연, 김성민, 정세환

## 권장 사용 흐름

1. 설정·이미지에서 기준 연도/월 선택
2. 근무표 이미지 업로드
3. 데이터 입력 탭에서 추출 프롬프트 복사
4. ChatGPT/Gemini에 이미지와 프롬프트 입력
5. 반환된 CSV를 데이터 입력 탭에 붙여넣기
6. CSV 불러오기
7. 데이터 확인 탭에서 결과 확인
8. 검수표에서 필요한 부분 수정
9. 클라우드 저장
