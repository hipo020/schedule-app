# 근무표 정리함 v1

이미지로 공유받은 월간 근무표를 보면서 사람별 스케줄 코드를 검수 입력하고, 내 스케줄과 오늘 출근 현황을 정리하는 정적 웹앱입니다.

## v1 기능

- 시트 탭형 페이지 이동
- 기준 연도/월, 내 이름, 확인 날짜 설정
- 스케줄표 이미지 업로드 및 미리보기
- 사람별 날짜 스케줄 코드 입력/검수
- 홈 요약
- 일간 / 주간 / 월간 보기
- 오늘 출근 현황
- 휴무일만 보기
- 카톡용 텍스트 복사
- 엑셀 다운로드
- 근무 코드 설정
- 브라우저 localStorage 저장

## Vercel 배포

1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. Vercel에서 Add New Project를 누릅니다.
3. 해당 GitHub 저장소를 선택합니다.
4. Framework Preset은 Other로 두거나 자동 감지 상태로 둡니다.
5. Build Command는 비워둡니다.
6. Output Directory는 `.` 로 설정합니다.
7. Deploy를 누릅니다.

## 사용 흐름

1. 설정·이미지 페이지에서 연도, 월, 내 이름, 확인 날짜를 설정합니다.
2. 스케줄표 이미지를 업로드해 원본을 확인합니다.
3. 검수표 페이지에서 이름과 날짜별 코드를 입력합니다.
4. 홈, 일간, 주간, 월간, 오늘 출근, 휴무일 페이지를 눌러 확인합니다.
5. 공유·엑셀 페이지에서 카톡용 텍스트를 복사하거나 엑셀 파일로 저장합니다.

## 참고

현재 버전은 OCR 자동 추출 전 단계의 검수형 MVP입니다. 다음 버전에서 이미지 자동 추출 기능을 추가할 수 있습니다.


## 이미지 업로드 후 사용 흐름

1. `설정·이미지` 페이지에서 이미지를 업로드합니다.
2. 업로드가 완료되면 미리보기와 `검수표 입력하러 가기` 버튼이 표시됩니다.
3. `검수표` 페이지에서 원본 이미지를 보며 사람 이름과 날짜별 근무 코드를 입력합니다.
4. 입력 후 `저장`을 누르면 홈, 일간, 주간, 월간, 오늘 출근, 휴무일, 공유·엑셀 페이지에서 정리된 결과를 확인할 수 있습니다.

현재 v1은 OCR 자동 추출 전 단계입니다. 이미지는 참고용으로 띄우고, 표 데이터는 검수표에서 직접 입력하는 방식입니다.


## 2026-06 저장 방식 수정

- 업로드한 이미지는 브라우저 저장공간 초과를 막기 위해 localStorage에 저장하지 않습니다.
- 이미지 업로드 후 탭 이동 중에는 그대로 유지됩니다.
- 새로고침하거나 브라우저를 닫은 뒤에는 이미지만 다시 업로드하면 됩니다.
- 스케줄 데이터, 코드표, OCR 결과는 계속 저장됩니다.
